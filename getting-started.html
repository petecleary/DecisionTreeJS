Get Started

Decision Tree gives you more control over the path your code takes to process an object.

There are some great linear flow control systems available, giving you asynchronous or parallel flow control. Async is one of the best.

However all offer linear processing with no decision making capabilities baked in.

Decision Tree is built to give you flow control based on flow chart priniciples.

By matching the common shapes in a flow chart the Decision Tree lets your code follow the rules based on the Operations and Decisions made in real time so each object you process can take a different path through the tree as required.

Quick Example

In the example below the flow will call the entry shape followed by an operation shape. The operation shape will add the value 2 to the data.a value. 

//create a data object to process
	var data = { "a": 1 };

	//create the root shape for the decision tree
	var shape = {
		"id": 1,
		"shapeType": "Entry",
		"description": "Start",
		"nextShape": {
			"id":2,
			"shapeType": "Operation",
			"description":"Adding value 2 to the data object",
			"nextShape": null,
			"properties": {"value": 2},
			"process": function (data, shape, callback) {
				data.a += Number(shape.properties.value);
				callback(null, shape);
			}
		}
	}
	
	//execute the tree on the root shape
    decisionTree.execute(shape, data, function(err, result, processedShapes){
        console.log("result: " + result.a);
        for (var x = 0; x < processedShapes.length; x++) {
            console.log("processed Shape: " + processedShapes[x].description + " id:" + processedShapes[x].id);
        }
    });

Working in Node.js

Running in Node.js is easy, simply include the library and then execute the tree. 


var decisionTree = require("decisionTree.min.js");
var TestApp = require("app.js");
var tree = require("json.js");

//create a data object to process
var data = { "a": 1 };

//execute the tree on the root shape
decisionTree.execute(tree.shape, data, TestApp.complete, { "TestApp": TestApp });

Review the examples to see more for Node.js.

Working in a browser

The decision tree works within the browser with the same syntax as the Node.js example. If you want you can make the execute call simpler by omitting any app libraries. 


<script type="text/javascript" src="decisiontree.js"></script>
<script type="text/javascript">

    decisionTree.execute(shape, data, function(err, result, processedShapes){
        console.log("result: " + result.a);
        for (var x = 0; x < processedShapes.length; x++) {
            console.log("processed Shape: " + processedShapes[x].description + " id:" + processedShapes[x].id);
        }
    });

</script>
